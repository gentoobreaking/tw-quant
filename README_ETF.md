# ETF 篩選系統 — 雙軌制 v4

針對台股 ETF 特性設計的自動化篩選工具，支援市值型、高股息、槓桿/反向型三類標的。

## 核心更新

- **自動成分股偵測**：優先從 yfinance 抓取即時前十大持股，並動態檢查成分股是否在 60MA 之上。
- **得分制體系**：專為 ETF 調優的加權評分（權重集中於趨勢與規模穩定性）。
- **資產規模排序**：優先篩選規模較大的 ETF，降低流動性風險。
- **穩定化分類**：精準判斷高股息門檻（4%），自動過濾非 TWD 計價標的。

## ETF 自動分類

| 類型 | 判斷邏輯 |
|------|---------|
| **槓桿/反向型** | 代號尾 L/R |
| **高股息** | `dividendYield` 穩定化判斷 ≥ 4% 或名稱含關鍵字 |
| **市值型** | 其餘標的 |

## 特色條件 (C5–C8)

- **C5 (溢價檢查)**: 透過 `navPrice` 計算溢價，< 0.5% 通過，負溢價 (折價) 額外加分。
- **C6 (成分股趨勢)**: 自動追蹤前 10 大成分股，需有 ≥ 2/3 站上 60MA。
- **C7 (特定門檻)**: 市值型費用率限制；高股息警示手動查詢項目。manual 旗標標記的條件不影響評分（假定通過），但會提示需手動確認。
- **C8 (高股息填息)**: 自動計算近 3 次除息後 30 日內填息率。資料不足時退回手動模式，不扣分。
- **C13 (非散戶接手)**: 含自營商判斷 — 法人賣超+融資增 或 自營商大買+融資增，均觸發散戶接警報（hard_reject）。

## Hard Reject 規則（觸發任一即直接 OUT）

| 規則 | 條件 | 意義 |
|------|------|------|
| C16 翻倍 | 市值型近40日漲≥80% / 高股息≥40% / 槓桿型近20日≥80% | 已過熱 |
| C17 高檔破月線 | 價位近年高 + 跌破20MA | 高檔反轉 |
| C20 空頭賣點 | 收<60MA + KD死叉>80區 | 趨勢轉空 |
| C13 散戶接手 | 法人賣超+融資增 或 自營商大買+融資增 | 散戶接刀 |

## Type-Adjusted 差異化門檻

槓桿/反向型與高股息 ETF 因波動特性不同，以下條件使用差異化門檻：

| 條件 | 市值型 | 高股息 | 槓桿/反向型 |
|------|--------|--------|-------------|
| C14 proximity_max | 5.0% | 5.0% | 10.0% |
| C15 連漲天數 | 3日 | 3日 | 近5日≥3日 |
| C16 gain_max | 80.0% | 40.0% | 80.0% (lookback=20) |

## 使用方式

執行 ETF 專屬篩選器：
```bash
python3 etf_screener.py
```

篩選結果會依「資產規模」由大到小排序輸出。

### 篩選結果解讀

- **C6（成分股趨勢）**：若顯示 `⚠️ 無成分股資料` 表示 Yahoo Finance 該 ETF 頁面尚無資料，屬正常現象（不影響整體評分）。
- **C7（市值型費用率）**：`netExpenseRatio` 直接取自 yfinance，需確認為近期數值。
- **C8（高股息填息自動檢查）**：自動計算近 3 次除息後 30 日內填息率，≥2/3 填息通過。無法計算時仍顯示 `⚠️` 需手動確認。

## 持股查詢工具

### `etf_top10_holdings.py`（快速互動查詢）

獨立腳本，直接爬 Yahoo Finance TW 頁面，輸入代號即可取得前十持股。

```bash
python3 etf_top10_holdings.py
# 輸入：0050
# 輸出：
# 0050 top 10 holdings:
#   ticker            name  weighting
#   2330        台積電        26.3%
#   2308          ...         ...
```

**用途**：快速查閱特定 ETF 的成分股及權重，無需等待完整篩選流程。

## 候選清單維護

候選清單由 `dump_candidates.py` 自動從 TWSE API 產出，無需手動維護。

```bash
# 查看新舊差異（不下載）
python3 dump_candidates.py --check

# 正式更新（直接寫入）
python3 dump_candidates.py
```

`candidates_ETF.csv` 現為 TWSE ETF 清單動態產生，與 `dump_candidates.py` 共同維護。新 ETF 掛牌後執行一次即可自動納入。

---

## 注意事項

- 部分槓桿型或反向型 ETF 無 NAV 資料，C5 會自動跳過。
- 高股息 ETF 的「填息」已自動化計算（C8），但「可分配收益」仍需依輸出警示手動確認。
- ETF 持股預取已優化：`screen_batch_prioritized()` 批次並行抓取前十持股（`ThreadPoolExecutor`），非逐檔 lazy 載入。
