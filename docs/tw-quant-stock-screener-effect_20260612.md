# stock_screener.py 效果說明

**日期**: 2026-06-12

## 一句話描述

對台股候選名單（1079 檔）逐一做 20 條件全景體檢，評分後分為 ENTER/WATCH/EXIT/OUT 四類，附帶出場信號偵測。

## 20 條件評分系統

| 類別 | 條件 | 權重 | 檢查內容 |
|------|------|------|----------|
| **技術面** | C1 | 10 | 股價 > 60MA + 季線上升 |
| | C2 | 5 | 股價 > 20MA |
| | C3 | 5 | 近5日低價曾觸及月線（回測站回） |
| | C4 | 10 | 量能 > 5日均量 |
| **基本面** | C5 | 5 | 溢價 < 3%（收盤偏離合理價） |
| | C6 | 5 | 市值分組後的 EPS/營收YoY/負債比達標 |
| | C7 | 5 | 存貨周轉天數未惡化 |
| | C9 | 5 | EPS > 分組門檻 |
| | C10 | 5 | 營收 YoY > 分組門檻 |
| **籌碼面** | C11 | 10 | 跌破均線時外資/投信連續買超 |
| | C12 | 10 | 千張大戶比率不減反增 |
| | C13 | 5 | 非散戶接手（法人大賣+融資增=hard_reject） |
| **位階** | C14 | 5 | 收盤距60MA 在 0~5%（中期位階） |
| | C15 | 5 | 連3日上漲（回測確認反轉） |
| | C16 | 5 | 近40日未漲≥80%（未翻倍） |
| | C17 | 10 | 非高檔跌破月線 |
| **買賣點** | C18 | 10 | 60分線黃金交叉 OR 日線帶量紅K |
| | C19_bonus | 5 | >60MA + KD黃金交叉（<20區）多頭買點加分 |
| | C20 | — | <60MA + KD死亡交叉（>80區）hard_reject |

**總分上限 100 分**，C19 是加分項（通過才加 5 分，不通過不扣）。

## 三層分類 + Hard Reject

| 分類 | 條件 | 意義 |
|------|------|------|
| 🟢 ENTER | 分數 ≥ 60 且無 hard_reject | 適合進場 |
| 🟡 WATCH | 分數 40~59 且無 hard_reject | 觀察名單 |
| 🔴 EXIT | 有出場信號（E1~E5） | 持有者應考慮出場 |
| ⚫ OUT | hard_reject 或分數 < 40 | 不建議 |

**Hard Reject 規則**（觸發任一即直接 OUT）：
- C16 翻倍（近40日漲≥80%）
- C20 空頭賣點（<60MA + KD死叉>80）
- C13 散戶接手（法人大賣+融資增）

## 出場信號（E1~E5）

| 信號 | 觸發條件 |
|------|----------|
| E1 | 前30日曾站上20MA → 現跌破月線 |
| E2 | 近高檔反轉 + KD死叉 |
| E3 | 跌破60MA + 60MA走平/向下 |
| E4 | 連跌4日 + 量縮<5MA 70% |
| E5 | 連3日量縮<5MA 60% + 價穩在月線上（主力退場前兆） |

## 執行方式

```bash
# 完整掃描（1079檔，約 30~60 分鐘）
python stock_screener.py

# 快速模式（跳過部分 API 呼叫）
python stock_screener.py --quick
```

輸出含：逐檔條件通過狀態、分類、出場信號、ENTER/WATCH 匯總表。
