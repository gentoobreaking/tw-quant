# TW-Quant 修改 4：C13 加入自營商判斷

**日期**: 2026-06-11~12
**狀態**: ✅ 完成

## 修改內容

C13（散戶接手）從只看外資+投信淨賣超+融資增，升級為：
1. **三大法人合計賣超**（加入自營商 T86 [11]）
2. **新增替代信號**：自營商大買+融資大增 → 疑似散戶透過自營商進場

## T86 欄位確認

TWSE fund/T86 API 19 欄位結構：
- **[4]** 外陸資買賣超（不含外資自營商）= 現有 `foreign_net`
- **[10]** 投信買賣超 = 現有 `trust_net`
- **[11]** 自營商買賣超（合計）= 新增 `dealer_net`
- [14] 自營商買賣超（自行買賣）
- [17] 自營商買賣超（避險）
- [18] 三大法人買賣超合計

用台積電驗證：外資+投信+自營商加總 = T86[18]，6 個日期完全吻合。

## C13 新邏輯

### 主邏輯（法人賣+融資增）
```
三大法人近 N 日合計賣超 + 融資餘額增加 → C13 散戶接手
明細顯示：哪些法人在賣（外資/投信/自營商）
```

### 替代信號（自營商買+融資增）
```
自營商近 N 日買超 > 0 且融資增加 且 自營商買超 > 融資增額 × 50%
→ 疑似散戶透過自營商避險帳戶進場
```

## 安全取值

若 T86 回傳欄位數 ≤ 11，跳過自營商（fallback 到 0），保持向後兼容：
```python
info["dealer_net"].append(p(11) if len(t86[stock_id]) > 11 else 0)
```
